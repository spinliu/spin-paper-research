# 论文科研全链路研究

围绕一个研究主题，提供从快速摸底到深度精读的全套工具链。

## 工具链概览

本 skill 按深度分为三个搜索层级，以及两个高级分析模块：

| 层级 | 工具 | 何时用 | 输出 |
|------|------|--------|------|
| L1 轻量 | 轻量级 web search | 快速了解主题、找关键词 | 相关论文列表 + 关键词 |
| L2 专业 | Paper Search（PubMed/arxiv/Crossref） | 精准定位论文、找高引论文 | 论文列表 + 元数据 + 摘要 |
| L3 深度 | Tavily AI Search | 深度调研、跨学科综述 | 结构化报告 + 来源分析 |
| L4 精读 | ljg-paper（论文精粹） | 单篇论文深度解读 | 结构化 Org 笔记 |
| L5 溯源 | ljg-paper-river（论文倒读法） | 理解论文脉络与演化 | 演化树 + 正向费曼叙事 |

## 核心工作流

### 首次接触一个研究主题

**步骤 1：L1 轻量搜索摸底（5 分钟）**

读取 `references/paper-tools.md` 中的「L1 轻量搜索」章节，执行快速主题摸底：
- 确定核心关键词（中英文）
- 识别该领域最重要的概念和术语
- 了解大致研究框架

**步骤 2：L2 Paper Search 精准定位（10-15 分钟）**

读取 `references/paper-tools.md` 中的「L2 专业论文搜索」章节，执行：
- 高引论文搜索（找该领域奠基性工作）
- 最新论文搜索（近 2-3 年进展）
- 按子方向分组整理

**步骤 3：L3 Tavily AI 深度调研（如需要）**

如果用户要求深度调研，读取 `references/paper-tools.md` 中的「L3 Tavily AI Search」章节，执行：
- 生成该主题的结构化调研报告
- 跨方向整合分析
- 识别研究空白和前沿趋势

**步骤 4：提示高级分析选项**

完成 L1-L3 后，提示用户：

```
已完成 {主题} 的文献摸底，找到 {N} 篇高引论文 + {M} 篇最新进展。

如需进一步深度分析，可选：

📄 论文精粹（ljg-paper）— 选任意 {N} 篇，我为你深度解读每篇的核心发现、方法要义和洞见

🌊 论文溯源（ljg-paper-river）— 选 1 篇，我倒着挖它的学术脉络，正着讲问题演化史

💬 两者都要 — 同时做精读 + 溯源
```

### 处理用户指定论文

当用户指定某篇/某几篇论文时：

1. **先判断用户意图**：精读（ljg-paper）还是溯源（ljg-paper-river）
2. **执行对应技能**：读取对应 integration 参考文件
3. **完成后提示其他方向**：告知可进一步做倒读法或精读其他论文

### 处理用户要求做"文献调研"

读取 `references/paper-tools.md`，按以下流程执行：

1. L1 确定关键词和主题边界
2. L2 找高引（奠基）+ 最新（前沿）
3. L3 生成结构化调研报告
4. 按子方向分组，列出每组代表论文
5. 提示可进一步精读或溯源的具体论文

## 输出格式规范

### 论文列表输出

```
## {主题} 文献调研

### 📊 基础数据
- 搜索关键词：{中英文关键词}
- 高引论文：{N} 篇（>20 citations）
- 最新论文：{N} 篇（近 2-3 年）

### 🏛️ 奠基性论文（高引）
| # | 论文 | 引用 | 核心贡献 |
|---|------|------|----------|
| 1 | ... | ... | ... |

### 🆕 最新进展
| # | 论文 | 年份 | 核心进展 |
|---|------|------|----------|
```

### 调研报告输出

包含：主题概述、关键研究问题、主流方法分类、各方向代表论文、当前研究空白、可关注的前沿方向。

## 注意事项

- 每次搜索都先明确研究主题和边界，避免发散
- L1/L2 搜索结果中如果发现相关但更精准的关键词，回代继续搜索
- 如果 Tavily API Key 未配置，提示用户配置后再执行 L3
- ljg-paper 和 ljg-paper-river 输出为 Org 文件存到 `~/Documents/notes/`
- 所有论文的元数据（PMID/arxiv ID/DOI）要保留，方便后续精读
